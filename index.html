Este manual foi elaborado para auxiliar na utilização dos recursos da Plataforma EmbryoAll.

Para começarmos, é importante que você conheça os principais botões e suas funcionalidades, após concluir este treinamento será muito mais simples entender as funcionalidades específicas para o seu perfil.

Principais Botões

Botão Cadastrar

Utilizado para cadastro de todas atividades de acordo com o ambiente acessado.

Ícone de edição

Para edição do documento correspondente.

Ícone lixeira

Para deletar o documento correspondente.

Lupa de pesquisa

Utilizado para realizar buscas específicas.

Impressora

Utilizado para impressão de arquivos em PDF.

Check

Utilizado para mudar o status da consulta.

Ícone atender

Acesso direto ao prontuário da paciente.

Ícone erlenmeyer

Visualização do prontuário de laboratório da paciente.

Ícone de visualização

Usado para ter uma visão geral de todo o banco de criopreservação.
