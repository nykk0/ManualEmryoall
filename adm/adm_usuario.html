Menu Usuários

O administrador irá cadastrar todos os usuários do sistema. Todos os botões (Administradores , Embriologistas, Enfermeiros, Equipe de orientação, Médicos e Secretárias) possuem a mesma funcionalidade, apenas o usuário é que muda. Para cadastro basta clicar no botão cadastrar, preencher as informações solicitadas e clicar em salvar.

Botão meus dados:

Onde você poderá alterar seus dados a qualquer momento.

Botão Log de Acesso:

Este botão é de extrema importância para controle de quem está acessando o sistema e de onde este acesso está acontecendo, qualquer acesso desconhecido, altere as senhas e nos comunique. A segurança de seus dados é uma prioridade da EmbryoAll.

Obs: Administrador após realizar o cadastro de seus usuários favor orientá-los a mudar suas senhas. Toda senha deve ser pessoal e intransferível para maior segurança das informações.
